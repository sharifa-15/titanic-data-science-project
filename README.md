# 📊 Titanic Survival Analysis Project

## 🔎 Overview
This project explores passenger survival patterns from the Titanic dataset through **data cleaning, analysis, and visualization**.  
It highlights how age, class, gender, and family size influenced survival outcomes.

## 🛠️ Tools Used
- **Python (Google Colab)** → Data cleaning & preprocessing  
- **Tableau** → Interactive dashboard design  
- **GitHub** → Project hosting & collaboration  

## 📂 Dataset
- Source: [Kaggle Titanic Dataset](https://www.kaggle.com/c/titanic/data)

## 📈 Key Insights
- Survival rates varied significantly by **class and gender**.  
- Younger passengers had higher survival chances compared to older groups.  
- Family size influenced survival probability in unexpected ways.
